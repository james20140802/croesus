**한 줄 결론.** 이 유니버스(현재 상장 521개 미국 주식)·기간(2016-06~2026-06, 121개 월별
cross-section)에서 가격 기반 공통 팩터의 cross-sectional 예측력은 **대체로 약하다**. 가장 강해
보이는 `beta_1y`·`volatility_3m`는 **사실상 동일한 "고위험=고수익" 베팅**(두 신호의 종목 간
순위상관 0.64)으로, 단일 강세장 + survivorship 편향의 산물일 가능성이 크며 저변동성/BAB
아노말리와 **반대 방향**이라 tradeable alpha로 신뢰하기 어렵다. 문헌과 정합적이고 비용·드로다운을
견디는 **유일하게 깨끗한 신호는 6개월 모멘텀(`momentum_6m`)**이며, 이마저 t≈2.1로 경계선이다.

## 1. Information Coefficient (Spearman, 시점별 → 시계열)

각 월별 cross-section에서 신호값과 h일 forward 수익률의 순위상관. `t_nw`는 Newey-West HAC t-stat
(forward 윈도 겹침을 시차 `round(h/21)`로 보정). `hit_rate`는 IC>0인 시점 비율. n≈105~111 시점.

| factor | h | mean IC | t_nw | IC IR | hit rate |
|--------|---|--------:|-----:|------:|---------:|
| momentum_1m | 21 / 63 / 126 | 0.005 / −0.000 / 0.011 | 0.37 / −0.02 / 1.00 | 0.03 / −0.00 / 0.08 | 52% / 44% / 60% |
| momentum_3m | 21 / 63 / 126 | 0.002 / 0.010 / 0.032 | 0.19 / 0.55 / **1.98** | 0.01 / 0.06 / 0.23 | 52% / 53% / 61% |
| **momentum_6m** | 21 / 63 / 126 | 0.010 / 0.025 / **0.042** | 0.78 / 1.48 / **2.08** | 0.05 / 0.17 / 0.30 | 53% / 57% / **70%** |
| volatility_3m | 21 / 63 / 126 | 0.034 / 0.063 / 0.083 | 1.64 / 1.94 / 1.92 | 0.14 / 0.27 / 0.38 | 55% / 61% / 69% |
| liquidity_1m | 21 / 63 / 126 | −0.007 / −0.017 / −0.026 | −0.83 / −1.25 / −1.25 | −0.07 / −0.17 / −0.26 | 44% / 39% / 42% |
| above_200d_ma | 21 / 63 / 126 | 0.009 / 0.015 / 0.021 | 0.80 / 0.95 / 1.18 | 0.06 / 0.12 / 0.17 | 55% / 56% / 62% |
| **beta_1y** | 21 / 63 / 126 | 0.043 / 0.079 / **0.101** | 1.82 / **2.06** / **2.08** | 0.15 / 0.29 / 0.41 | 57% / 58% / 66% |

**|t_nw|>2인 셀**: `beta_1y`(63, 126), `momentum_6m`(126), 그리고 `momentum_3m`(126, t=1.98 경계).
다중검정: 7팩터×3horizon = **21개 검정** → |t|>2는 우연히도 ~1개 기대. 관측된 유의 셀은 상관 높은
신호(beta≈vol)와 최장 horizon에 몰려 있어 **실질 독립 발견은 ~2개**(위험/베타 베팅 1 + 모멘텀 1),
우연 기대치보다 아주 조금 높은 수준이다.

## 3. 왜 vol/beta를 믿으면 안 되는가 (자기기만 방지의 핵심)

1. **같은 베팅.** `beta_1y`와 `volatility_3m`의 시점별 종목 순위상관 = **0.637**. Q5−Q1 롱숏은
   "고베타 롱 − 저베타 숏" = **순-롱-마켓 레버리지** 포지션. 2016~2026 강세장에서 이런 포지션은
   기계적으로 돈을 번다 — 이건 시장 프리미엄을 레버리지로 수확한 것이지 시장중립 alpha가 아니다.
2. **문헌과 반대.** 잘 문서화된 저변동성/BAB(betting-against-beta) 아노말리는 **저위험이
   위험조정 기준으로 이긴다**고 본다. 우리 결과는 정반대 → 특정 표본/레짐 아티팩트 신호.
3. **Survivorship이 정확히 여기를 때린다.** `assets`는 현재 상장 종목만 담는다. 폭발한 고베타
   종목(상장폐지·인수·파산)이 빠지고 살아남아 급등한 고베타 종목만 남아 → 고베타 성과가 상방편향.
   survivorship 편향은 고위험 구간에서 가장 크다.
4. **단일 레짐.** 121개월 전부 대체로 완화적·강세 국면. 고베타 우위는 2020·2022 급락에서 역전됐을
   가능성이 크며, 표본이 그 꼬리를 충분히 담지 못한다.

→ **결론: vol/beta의 높은 Sharpe는 tradeable edge가 아니라 함정.** 반대로 `momentum_6m`은
beta/vol과 직교하고, drawdown이 낮으며, IC가 horizon에 따라 단조 증가(0.010→0.025→0.042)하는
문헌 정합적 패턴 → **유일하게 추적할 가치가 있는 신호**(그래도 t≈2.1, 경계선).

## 4. 순열검정 주의

시점별 신호 셔플 귀무(h=63)의 z-score는 크다(beta 19.6, vol 15.5, momentum_6m 6.3). 그러나 이
검정은 **각 날짜 cross-section을 독립으로 취급**해 시계열 자기상관·레짐 지속성을 무시하므로 유의성을
과대평가한다. 시간 변동을 반영하는 **Newey-West t-stat(~2)이 정직한 값**이다. 두 검정의 큰 간극
자체가 "신호가 시간에 걸쳐 얼마나 불안정한가"를 말해준다.

## 5. 자기기만 방지 체크리스트 (이번에 실제로 걸린 것 포함)

- **[실제로 걸림] 중첩 수익률 복리.** 초기 구현이 6개월 forward 수익률을 매월 복리해 누적수익을
  ~6배(예: +1,600,000%) 과대계상. 비중첩 보유기간 백테스트로 교정. (`run.py` 방법론 주석 참조.)
- **다중검정.** 21개 검정, |t|>2 우연 기대 ~1개 병기. 순열·decay 일관성으로 교차확인.
- **Survivorship.** 현재 상장분만 → 상방편향, 특히 고베타. §3에 명시.
- **Look-ahead 없음.** 팩터는 `[:as_of]`, forward 수익률은 `[as_of:as_of+h]`만 사용.
- **거래비용.** 0/10/20bps 민감도 보고. 저회전 신호(vol/beta/liquidity)는 비용에 강하고,
  고회전(momentum_1m)은 비용에 취약.
- **adjusted_close 편차.** Croesus는 `close` 사용; 여기선 배당조정 total-return 위해 adjusted 사용.
- **등가중 소형주.** 분위 등가중은 소형주 비중이 커 실거래 유동성/비용은 표보다 불리할 수 있음.
